# autoinstaller
Поможет вам установить дополнительные программы

Termius connecter phone:
1) На телефон установите приложение Termius (app store, google play)
2) Зайдите в приложение от имени администратора
3) Выберите свою ос и нажмиете Installer
4) Выберите Termius connecter phone
5) Дождитесь установки всех файлов
6) Когда вы получите ssh username@yourIP зайдите в мобильное приложение (termius)
7) Перейдите в вкладку connections
8) В поле ввода введите ssh username@yourIP
9) если попросит порт > 22. Если попросит пароль > введите пароль от компьютера
10) после того как все выйдет у вас появится вкладка с названием username@yourIP
11) готово
